# OnionReel Production Lifecycle Checklists v1

## Pre-Production Checklist

### When to Use
Use this checklist during the initial planning phase of a film, advertisement, or media project.

### Inputs
- Project brief
- Creative brief
- Budget
- Timeline

### Outputs
- Finalized script
- Storyboards
- Production plan
- Asset list

### Steps
- [ ] **Brief Intake**
  - Gather project requirements and objectives.
- [ ] **Concept Development**
  - Brainstorm and outline the core idea.
- [ ] **Scriptwriting**
  - Write and revise the script.
- [ ] **Storyboard Creation**
  - Create visual representations of scenes.
- [ ] **Shot List Preparation**
  - List all shots needed for production.
- [ ] **Asset Planning**
  - Identify and list all assets (props, costumes, locations).
- [ ] **Production Planning**
  - Develop a detailed production schedule and budget.

### Acceptance Criteria
- All stakeholders approve the script and storyboard.
- Production plan aligns with budget and timeline.

### Common Failure Modes + Fixes
- **Failure Mode:** Script revisions cause delays.
  - **Fix:** Set a strict deadline for final script approval.
- **Failure Mode:** Missing assets during production.
  - **Fix:** Conduct a thorough asset inventory before shooting.

## Production Checklist

### When to Use
Use this checklist during the actual filming or recording phase.

### Inputs
- Final script
- Shot list
- Production plan

### Outputs
- Raw footage
- Audio recordings
- Production notes

### Steps
- [ ] **Camera Setup**
  - Set up cameras according to shot list.
- [ ] **Lighting Setup**
  - Adjust lighting for each scene.
- [ ] **Audio Setup**
  - Ensure microphones and audio equipment are ready.
- [ ] **Data Wrangling**
  - Organize data storage for footage.
- [ ] **Slate/Naming**
  - Use slates to label shots and take numbers.
- [ ] **Continuity Notes**
  - Document details to maintain continuity.

### Acceptance Criteria
- All planned shots are captured.
- Audio is clear and usable.

### Common Failure Modes + Fixes
- **Failure Mode:** Camera or audio equipment failure.
  - **Fix:** Have backup equipment on hand.
- **Failure Mode:** Inconsistent lighting.
  - **Fix:** Use reference images to maintain consistency.

---

## Post-Production Checklist

### When to Use
Use this checklist after filming is complete, during the editing and finalization phase.

### Inputs
- Raw footage
- Audio recordings
- Production notes

### Outputs
- Final edited film/advertisement
- Exported files
- Quality control report

### Steps
- [ ] **Ingest Footage**
  - Import all footage and audio into editing software.
- [ ] **Assembly Edit**
  - Create a rough cut of the project.
- [ ] **Pacing Adjustments**
  - Refine the flow and timing of scenes.
- [ ] **Sound Design**
  - Edit and mix audio tracks.
- [ ] **Color Correction**
  - Adjust colors for consistency and mood.
- [ ] **Captioning**
  - Add captions for accessibility.
- [ ] **Final Exports**
  - Export the final project in required formats.
- [ ] **Quality Control (QC)**
  - Review the final product for errors.

### Acceptance Criteria
- Final product meets creative vision and technical standards.
- All QC issues resolved.

### Common Failure Modes + Fixes
- **Failure Mode:** Missing or corrupted files.
  - **Fix:** Regularly back up all data during production.
- **Failure Mode:** Inconsistent audio levels.
  - **Fix:** Use audio meters to ensure balanced levels throughout.

--- 

This checklist is designed to streamline the production lifecycle for films, ads, and media projects, ensuring that all critical steps are followed and common pitfalls are addressed.
