# OnionReel Remotion

This folder contains the Remotion-based render engine.

## Install

```bash
cd onionreel/remotion
npm install
```

## Render (dev)

```bash
npm run remotion:preview
```

## Render (CLI)

```bash
npm run remotion:render -- Reel30 out.mp4
```

Compositions are defined in `src/Root.tsx`.
