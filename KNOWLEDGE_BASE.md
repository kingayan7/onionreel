# OnionReel Knowledge Base (living doc)

## Taxonomy
### Pre-production
- Creative brief → concepting
- Script writing
- Storyboarding
- Shot list + production plan
- Casting/locations/props

### Production
- Camera + lighting setups
- Audio capture
- On-set workflow + data wrangling

### Post-production
- Editing (pacing, structure)
- Sound (dialogue, SFX, music, mix)
- Color grading
- VFX/motion
- Subtitles + deliverables

### AI Tooling
- Image generation (references, storyboards)
- Video generation (shots, b-roll, transitions)
- Voice (VO, ADR)
- Music/SFX
- Upscale, frame interpolation

### Quality system (the moat)
- Checklists
- Rubrics
- Common failure modes + fixes

## Ingestion rules
- Prefer: SOP/checklist format with concrete steps and acceptance criteria.
- Every entry should include: "When to use", "Inputs", "Outputs", "Steps", "Pitfalls".
