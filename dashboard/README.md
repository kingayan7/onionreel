# OnionReel Dashboard (MVP)

This folder is the first runnable OnionReel dashboard module.

## Run

From the OnionReel repo root:

```bash
cd dashboard
node server.mjs
```

Open: http://127.0.0.1:5057

## API

- `GET /api/roadmap` → returns `../CONTINUOUS_BUILD_ROADMAP.json`
